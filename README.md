# DownYoutube
 A GUI Downloader for downloading songs and videos from youtube alone. It has rich user interface compiled from SCSS using Grid and Flex layout and also coming with animation.Check out here[JDownloader](https://github.com/janakhpon/DownYoutube.git).



# Usage

clone the repository

    git clone https://github.com/janakhpon/DownYoutube.git

Get into root folder

    cd DownYoutube

Install Dependencies using [npm](https://www.npmjs.com/)

    npm install
    npm i -d
    npm start

Install Dependencies using [yarn](https://yarnpkg.com/en/)

    yarn install
    yarn add
    yarn start




# Warning
If u run into error use the following steps

```bash
sudo yarn install
sudo yarn add
sudo yarn start
```
or

```bash
sudo npm install
sudo npm i -d
sudo npm start
```
